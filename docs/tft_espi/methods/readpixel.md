## Description

Read 565 pixel colours from a pixel.

## Signature

`uint16_t TFT_eSPI::readPixel(int32_t x0, int32_t y0)`

## Parameters

* `int32_t` x0: x coord;
* `int32_t` y0: y coord.

## Result

`uint16_t`: the [color](../colors.md) in 565 bits.
