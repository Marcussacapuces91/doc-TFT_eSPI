## Description

Set the text cursor x,y position or the text cursor x,y position and font.

## Signature

* `void TFT_eSPI::setCursor(int16_t x, int16_t y)`
* `void TFT_eSPI::setCursor(int16_t x, int16_t y, uint8_t font)`

## Parameters

* `int16_t` x: x coord.
* `int16_t` y: y coord.
* `uint8_t` font: Font number.

## Result

None.
